== Überwatch Source Code

This is the source code for Überwatch.

Here's a quick overview:

* Ruby 2.3.1
* Rails 4.2.6

Additional gems:

* Sorcery for user authentication & encryption (link: https://github.com/NoamB/sorcery)
* Kaminari for pagination (link: https://github.com/amatsuda/kaminari)
* pg_search for search methods (link: https://github.com/Casecommons/pg_search)
* acts_as_votable for upvoting/downvoting (link: https://github.com/ryanto/acts_as_votable)
